# Fastify Starter

## Environment variables

```sh
- NODE_ENV................ [development|production] default 'development'
- COOKIE_SECRET........... Base64 random string default <none>, **required**
- API_HOST................ Listening host FQDN/IP default 'localhost'
- API_PORT................ Listening IP Port default 3000
- TIMEZONE................ default 'Asia/Bangkok'
```
